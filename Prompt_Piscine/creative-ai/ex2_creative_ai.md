# Exercise 2: Personalized Resume Summary

## Write a prompt that generates a resume summary based on role and skills. Example: "Create a 3-sentence professional summary for a software engineer skilled in Python and cloud systems."

## **Prompt:**
Create a 3-sentence professional summary for a software engineer skilled in JavaScript and cloud systems.

## **AI Model:**

Results-driven Software Engineer with expertise in JavaScript and modern cloud systems, delivering scalable and efficient web applications. Experienced in designing, deploying, and maintaining cloud-based solutions while ensuring performance, security, and reliability. Passionate about leveraging cutting-edge technologies to solve complex problems and drive business value.


## Adjust the prompt for a different role (e.g., teacher, designer, marketer).

## **Prompt:**
Create a 3-sentence professional summary for a farmer skilled in farm management and agribusiness

## **AI Model:**
Experienced Farmer with expertise in farm management and agribusiness, skilled in optimizing crop production and resource utilization. Proficient in implementing sustainable farming practices, improving yield, and reducing post-harvest losses. Passionate about integrating innovative techniques and business strategies to grow profitable and environmentally responsible agricultural ventures.

## Compare how the outputs change depending on the role.

The role shapes professional summaries by influencing content, style, and tone, a software engineer summary emphasizes technical skills, cloud systems, and problem solving, written in a formal tone, while a farmer summary highlights farm management, agribusiness, and sustainability.
